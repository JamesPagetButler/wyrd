# Security policy

## Supported versions

Wyrd is in Crawl phase (v0.1.0-alpha). No production deployments yet.
Security reports against the v0.1.x line will be triaged but fixes may
land on `main` only.

## Reporting a vulnerability

For security issues:

1. **Do NOT open a public issue.**
2. Email the maintainer privately, or use GitHub's private security
   advisory feature on this repository.

We will acknowledge receipt within 72 hours. If the issue is in the Lean
corpus (a soundness gap, a missing assumption, or a counterexample to a
posted theorem), please describe it precisely enough to reproduce in
`lake build`.
